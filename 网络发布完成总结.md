## 🚀 发布步骤

### 第一步：推送到 GitHub
```bash
# 1. 初始化并推送代码
git init
git add .
git commit -m "CNMoney v1.0.0 - 多货币管理插件首次发布"
git remote add origin https://github.com/你的用户名/cnmoney.git
git push -u origin main

# 2. 或使用自动化脚本
发布.bat
```

### 第二步：创建 GitHub Release
1. 访问 GitHub 仓库
2. 创建 Release：标签 `v1.0.0`
3. 填写发布说明

### 第三步：JitPack 构建
1. 访问 [JitPack.io](https://jitpack.io)
2. 输入：`你的用户名/cnmoney`
3. 选择版本：`v1.0.0`
4. 等待构建完成

## 📦 使用方式

### Maven 依赖配置

其他开发者可以这样使用：

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>com.github.你的用户名</groupId>
        <artifactId>cnmoney</artifactId>
        <version>v1.0.0</version>
        <scope>provided</scope>
    </dependency>
</dependencies>
```

### plugin.yml 配置
```yaml
depend: [CNMoney]
```

### Java 代码调用
```java
// 获取 API
CNMoneyAPI api = CNMoney.getAPI();

// 基本操作
BigDecimal balance = api.getBalance(player, "gold_coin");
api.addBalance(player, "gold_coin", new BigDecimal("100"));
boolean success = api.removeBalance(player, "gold_coin", new BigDecimal("50"));
```
